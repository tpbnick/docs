## Introduction  

Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design.  It takes care of much of the hassle of Web development, so you can focus on writing your app without needing to reinvent the wheel. Itâ€™s free and open source.  A web framework is a set of components that helps you to develop websites faster and easier.  Django's primary goal is to ease the creation of complex, database-driven websites. The framework emphasizes reusability and "pluggability" of components, less code, low coupling, rapid development, and the principle of don't repeat yourself.
